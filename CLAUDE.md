# 项目要求

使用 pretext 框架创建一个精美的书页网页应用。

## 要求
1. 在当前目录创建项目
2. 使用 pretext 库 (@chenglou/pretext) 进行文本布局
3. 设计一个类似真实书页的界面，包含：
   - 优雅的页面布局（页边距、页眉、页脚）
   - 精美的中文字体支持
   - 可翻页效果或滚动阅读体验
   - 响应式设计
4. 使用示例文本（中文文章或诗歌）
5. 部署到 GitHub Pages 或 Vercel，提供公网访问链接

## 技术栈
- 使用 Bun 作为包管理器
- 使用 pretext (@chenglou/pretext) 进行文本测量和布局
- 使用原生 HTML/CSS/JS

## 输出
完成后提供部署的公网 URL
