✨ Pretext 文本布局引擎

纯 JavaScript/TypeScript 多语言文本测量与布局库
快速、精准、支持所有你想得到的语言

🚀 GitHub 仓库
⚡

零 DOM 依赖

无需触发重排重绘，纯算术计算文本高度，性能提升 100 倍+

const prepared = prepare('文本', '16px Inter')
const { height } = layout(prepared, width, lineHeight)
// 纯算术，无 DOM 操作！
🌍

全语言支持

中文、英文、阿拉伯语、emoji、混合双向文本，全部完美支持

prepare(
  'AGI 春天到了 🌸' +
  'بدأت الرحلة 🚀'
)
📐

精准布局

使用浏览器字体引擎作为基准，AI 友好的迭代测量方法

// prepare: ~19ms (500 文本批量)
// layout: ~0.09ms (同批量)
const { lines } = layoutWithLines(...)
🎨

多场景渲染

支持 DOM、Canvas、SVG、WebGL，甚至服务端渲染

for (line of lines) {
  ctx.fillText(line.text, 0, y)
  y += lineHeight
}
📱

响应式布局

动态计算容器宽度，自动换行，完美适配各种屏幕

window.addEventListener('resize', () => {
  layout(prepared, newWidth, lineHeight)
})
🔍

高级 API

walkLineRanges、layoutNextLine 等高级功能，满足复杂需求

walkLineRanges(prepared, width,
 line => {
  if (line.width > maxW)
    maxW = line.width
 })
