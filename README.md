# 📖 书页阅读 - Pretext 演示

一个使用 Pretext 文本布局引擎打造的精美书页网页应用。

## 在线演示

**[https://nodiff.github.io/pretext-book-page](https://nodiff.github.io/pretext-book-page)**

## 特性

- 📜 **精美书页设计** - 仿纸质书籍的视觉体验
- 🎨 **优雅排版** - 使用中文字体，支持传统阅读习惯
- 📱 **响应式设计** - 适配手机、平板、桌面设备
- 🔄 **翻页效果** - 流畅的页面切换动画
- ⚡ **Pretext 引擎** - 纯 JavaScript 文本测量和布局

## 技术栈

- **Pretext** (@chenglou/pretext) - 文本测量和布局引擎
- **原生 HTML/CSS/JS** - 零框架依赖
- **ES Modules** - 现代化模块系统

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 部署到 GitHub Pages
npm run deploy
```

本项目使用李白的《春夜宴桃李园序》作为示例文本，包含原文和译文。
